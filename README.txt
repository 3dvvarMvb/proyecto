Proyecto de Gestión de Eventos de Tráfico

Este proyecto recopila, almacena y gestiona eventos de tráfico utilizando múltiples servicios en contenedores Docker.

Requisitos Previos

1. Tener instalado Docker y Docker Compose.
2. Asegurarse de que los puertos necesarios (por ejemplo, 5000) estén disponibles.

Configuración Inicial

1. Clonar el repositorio en tu máquina local.
2. Navegar al directorio raíz del proyecto.

Instrucciones para Ejecutar el Proyecto

Construir y levantar los servicios con Docker Compose ingresar en terminal:

   sudo docker-compose up --build
